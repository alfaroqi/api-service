#### Media Endpoint

|  METHOD  |       API       |        REMARKS         |
| :------: | :-------------: | :--------------------: |
|  `POST`  |     /media      |         Create         |
|  `GET`   |     /media      |        Get All         |
| `DELETE` | /media/:mediaId | Delete Data By mediaID |
